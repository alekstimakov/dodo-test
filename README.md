# dodo-test

Репозиторий содержит прототип детекции состояний столика по видео.

Основная рабочая версия находится в папке [`1`](./1):

- код: `1/main.py`
- инструкция: `1/README.md`
- зависимости: `1/requirements.txt`
- артефакты: `1/output.mp4`, `1/events.csv`

## Быстрый запуск

```bash
cd 1
pip install -r requirements.txt
python main.py --video video1.mp4
```

Если нужно полное описание логики, параметров и ТЗ, смотрите:

- [`1/README.md`](./1/README.md)
