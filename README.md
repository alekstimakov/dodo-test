# Прототип детекции уборки столиков (1 видео)

## Запуск
```bash
pip install -r requirements.txt
python main.py --video video1.mp4 --output output.mp4
```

Опционально:
```bash
python main.py --video video1.mp4 --output output.mp4 --events-csv events.csv
```

## Какое видео и столик выбраны
- Видео: `video2.mp4` (одно тестовое видео).
- Столик: выбирается вручную на первом кадре через `cv2.selectROI` (прямоугольник).

## Логика детекции событий
- Детектор: `YOLOv8n` (`ultralytics`).
- Анализируется один выбранный столик + расширенная `near`-зона вокруг него.
- События:
  - `EMPTY`: в зоне нет детекций-триггеров.
  - `APPROACH`: детекция появилась в `near`-зоне или как промежуточный переход `EMPTY -> OCCUPIED`.
  - `OCCUPIED`: детекция находится в зоне столика.
- Для защиты от дребезга смена состояния подтверждается через таймаут (`state_hold_seconds`).

## Аналитика
- Все события пишутся в `events.csv` (`frame`, `time_sec`, `event`).
- Для каждого `EMPTY` ищется следующий `APPROACH`.
- Считается метрика: среднее время между `EMPTY` и следующим `APPROACH`.
- Метрика печатается в консоль после обработки видео:
  - `Average delay: XX.XX sec`
